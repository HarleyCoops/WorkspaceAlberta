## Connected Tools

This workspace is configured to connect with:

- **Google Drive** - Files, Docs, Sheets, Slides
- **Slack** - Channels, messages, and DMs
- **Buffer** - Social posting
- **Mailchimp** - Email marketing
- **Google Analytics** - Web and product analytics
- **Calendly** - Scheduling links and bookings

## Adding API Keys

Some tools require API keys. Add them in your Codespaces secrets:

1. Go to github.com/settings/codespaces
2. Click "New secret"
3. Add the required keys (see `docs/WELCOME.md` for details)
