# Google Drive AI Workspace

A pre-configured workspace that connects your Google Drive to AI assistants.

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/HarleyCoops/GoogleDriveWorkspace?quickstart=1)

## What This Does

This workspace sets up everything you need to:
- Connect Google Drive to your AI assistant
- Search, read, and organize your Drive files using natural language
- Ask questions about documents without opening them

## Quick Start

1. **Click the badge above** to create your workspace
2. **Follow SETUP-GUIDE.md** (opens automatically)
3. **Run `python scripts/oauth_setup.py`** to authorize Google Drive
4. **Start chatting** with the AI about your files!

## Example Conversations

Once set up, you can ask the AI things like:

- "List my most recent documents"
- "Find all spreadsheets from last month"
- "What's in the file called 'Budget 2024'?"
- "Create a new folder for project X"

## Files in This Workspace

| File | Purpose |
|------|---------|

| `.vscode/mcp.json` | MCP server configuration |

## Requirements

- A Google account
- A GitHub account (free)
- 10-15 minutes for initial setup

## Privacy

- Your Google credentials are stored in YOUR GitHub account
- The workspace owner cannot see your credentials or files
- All data stays in your Codespace
